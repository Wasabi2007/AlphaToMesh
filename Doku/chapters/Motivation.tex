\chapter{Motivation}

Wenn Sprites mit einem großen Transparenten Flächen in Spielen verwendet werden, wird geht viel Rechenzeit für die Berechnung und Verwerfung von Absolute transparenten Pixeln drauf. In solchen fällen kann es Besserer sein einen Angepassten Mesh zur Textur zu verwenden anstelle eines einfachen Quadrates.
Außerdem kann der Mesh als Kollisionsgeometrie benutzt werden wenn per Pixel Kollision zu aufwendig ist.
